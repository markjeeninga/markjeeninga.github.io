  • In the paper “Braess’ paradox for power flow feasibility and parametric uncertainties in DC power grids with constant-power loads” it was shown that the topologies of most MATPOWER cases, when interpreted as a DC power grid, are subject to Braess’ paradox for power flow feasiblity. The code can be found in this GitHub repository. The code is written in Julia and makes use of the PowerModels package
